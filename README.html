AI4LAM Metadata Working Group#

Meetings, Projects, and Notebooks#

The AI4LAM Metadata Working Group Jupyter Book contains meeting notes, projects and Jupyter notebooks.

From our charter:

The Metadata Working Group will explore and share the ways in which AI
technologies can be applied to the creation and enhancement of metadata for cultural heritage resources. It will also address the role of metadata in collecting, curating, and describing the resources needed for AI, machine learning, and data science more broadly.

Jupyter Notebooks, data, and associated source code for interactive sessions in the AI4LAM Metadata Working Group’s monthly meeting.

Our intention is allow attendees to access a shared Jupyter Notebooks hosted on Binder.
